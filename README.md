Flappy bird in Python with doom theme
=====================
[Youtube Video](https://www.youtube.com/watch?v=RR8F-FzlCgU)

![Alt text](/screenshot.png?raw=true "flappydoom")

Requirements
------------
```bash
pip install pygame
```

To play
-------
1. Clone the repo.
2. `python main.py`
3. Use left mouse click
